HumanoidController

Scripting

To utilize this script, it cannot be directly attached to a game object as it functions as an abstract class. Instead, you must create a new class that inherits from HumanoidController rather than MonoBehaviour. Once your custom script is prepared, you can then attach it to your character object.

Note that the HumanoidController requires some components in order to work properly. Those would be the following:
CharacterMovement,
CharacterEnvironmentalQuery,
Rigidbody2D

Attaching the script to your character object will also automatically attach the referenced script as well.
