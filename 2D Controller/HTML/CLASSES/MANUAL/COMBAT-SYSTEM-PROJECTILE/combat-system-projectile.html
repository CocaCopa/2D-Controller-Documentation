CombatSystemProjectile

SWITCH TO SCRIPTING

Overview

The CombatSystemProjectile component is crafted to handle the behaviour of any object, meant to act as a projectile in your game.

To utilize the component in your Unity project, proceed by attaching the script to the root object of the projectile in the Unity Editor, and then configure the parameters exposed in the Inspector to customize its behaviour as per your requirements.

Note that the CombatSystemProjectile requires a Rigidbody2D component for proper operation. Attaching the script to your projectile object will automatically add this required component.


Inspector


Speed

InitialVelocity The initial velocity of the projectile when it's launched.
MinimumInitialVelocity If the 'initialVelocity' of the projectile is modified by an external script, it is ensured that the new value will not fall below the specified minimum.

Damage

DamageAmount The amount of damage the projectile should deal upon colliding with a damageable object.
MinimumDamageAmount If the 'damageAmount' of the projectile is modified by an external script, it is ensured that the new value will not fall below the specified minimum.

Hitbox

HitboxTranform The transform of the hitbox object attached to the projectile object.
VisualizeHitbox Visualizes shape and size of the hitbox in the scene.
HitboxShape The shape of the hitbox.
HitboxRadius The radius of the hitbox.
