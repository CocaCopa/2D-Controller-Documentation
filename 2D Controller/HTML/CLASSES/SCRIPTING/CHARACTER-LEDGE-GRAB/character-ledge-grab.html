CharacterLedgeGrab

Dependencies

Required Component: Rigidbody2D


Description

This class will help you manage your character Ledge Grab and/or Ledge Climb state.


Public Methods
EnterLedgeGrabState Sets your character's state to Ledge Grab.
ExitLedgeGrabState Exit from Ledge Grab state.
LedgeClimb Makes the character climb at the top of a ledge.
