CharacterCombat.CancelChargeAttack

Declaration

public void CancelChargeAttack(AttackSO attackData);


Parameters
attackData The scriptable object (AttackSO) that holds the data of the attack.

Description

Cancels the current charge attack.

Upon invoking the PerformChargeAttack function, the ongoing charge for the attack can be canceled by calling this function. If the attack is set to automatically cancel when held for a duration surpassing the specified time (HoldAttackTimer), the PerformChargeAttack function will trigger this cancellation automatically. However, you can call this function manually if a certain condition is met.


Here is an example of how you can use this function. 

using UnityEngine;
using CharacterController2D.CombatSystem;

[RequireComponent(typeof(CharacterCombat))]
public class Example : MonoBehaviour
{
    [SerializeField] private AttackSO exampleAttack;
    [SerializeField] private Transform projectileSpawnPoint;
    
    private CharacterCombat characterCombat;

    private void Awake()
    {
        characterCombat = GetComponent<CharacterCombat>();
    }

    private void Update()
    {
        if (Input.GetKey(KeyCode.X))
        {
            characterCombat.PerformChargeAttack(exampleAttack, projectileSpawnPoint);

            if (Input.GetKeyDown(KeyCode.C))
            {
                characterCombat.CancelChargeAttack(exampleAttack);
            }
        }
        else if (Input.GetKeyUp(KeyCode.X))
        {
            characterCombat.TryReleaseChargeAttack(exampleAttack);
        }
    }
}
