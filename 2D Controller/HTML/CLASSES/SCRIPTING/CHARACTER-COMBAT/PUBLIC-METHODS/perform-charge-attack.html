CharacterCombat.PerformChargeAttack

Declaration

public void PerformChargeAttack(AttackSO attackData, Transform projectileSpawnPoint = null);


Parameters
attackData The scriptable object (AttackSO) that holds the data of the attack.
projectileSpawnPoint If the attack throws a projectile, a transform reference is needed.

Description

Starts charging the specified charge attack.

This function initiates the charging process for an attack, adhering to a predefined duration specified by the attack's configuration. If this set duration is surpassed, the character is compelled to either cancel or release the attack. The amount of charge accumulated during the charging period is stored in seconds within the ChargeTimer property.


Here is an example of how you can call this function. A reference to an AttackSO scriptable object is required. 

using UnityEngine;

[RequireComponent(typeof(CharacterCombat))]
public class Example : MonoBehaviour
{
    [SerializeField] private AttackSO exampleAttack;
    
    private CharacterCombat characterCombat;

    private void Awake()
    {
        characterCombat = GetComponent<CharacterCombat>();
    }

    private void Update()
    {
        if (Input.GetKey(KeyCode.X))
        {
            characterCombat.PerformChargeAttack(exampleAttack);
        }
    }
}


In the given example, the function is employed to execute a chargeable attack that also includes the launching of a projectile. Additionally, a reference to a transform point is necessary for the attack to utilize it as a spawn point for the projectile. 

using UnityEngine;

[RequireComponent(typeof(CharacterCombat))]
public class Example : MonoBehaviour
{
    [SerializeField] private AttackSO exampleAttack;
    [SerializeField] private Transform projectileSpawnPoint;
    
    private CharacterCombat characterCombat;

    private void Awake()
    {
        characterCombat = GetComponent<CharacterCombat>();
    }

    private void Update()
    {
        if (Input.GetKey(KeyCode.X))
        {
            characterCombat.PerformChargeAttack(exampleAttack, projectileSpawnPoint);
        }
    }
}
