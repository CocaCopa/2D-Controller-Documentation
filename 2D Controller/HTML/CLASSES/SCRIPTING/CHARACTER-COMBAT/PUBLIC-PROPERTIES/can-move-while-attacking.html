CharacterCombat.CanMoveWhileAttacking

Declaration

public bool CanMoveWhileAttacking { get; }


Description

Indicates whether the character is able to move while attacking.

Depending on the configuration of the attack's AttackSO scriptable object, the character's movement can be restricted. If the attack is configured to disallow movement while attacking, you need to consider this in your custom controller's script and constrain the character's movement by reading the value of the corresponding property.
It's worth noting that the HumanoidController class is already designed to handle such restrictions.


Here is an example of how you can access the property. 

using UnityEngine;

[RequireComponent(typeof(CharacterCombat))]
public class Example : MonoBehaviour
{
    [SerializeField] private AttackSO exampleAttack;
    
    private CharacterCombat characterCombat;

    private void Awake()
    {
        characterCombat = GetComponent<CharacterCombat>();
    }

    private void Update()
    {
        if (Input.GetKey(KeyCode.X))
        {
            characterCombat.PerformNormalAttack(exampleAttack);
            Debug.Log("Can move while attacking: " + characterCombat.CanMoveWhileAttacking);
        }
    }
}
