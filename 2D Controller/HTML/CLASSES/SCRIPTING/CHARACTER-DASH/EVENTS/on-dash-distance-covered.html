CharacterDash.OnDashDistanceCovered

Declaration

public event EventHandler<OnDashDistanceCoveredEventArgs> OnDashDistanceCovered;


Event Arguments
targetDashPosition The claculated position the character should be at, once the dash is completed.

Description

Triggered when the dash action has been completed.

Subscribe to this event to be notified when your character has completed a dash. You can utilize the event's argument to adjust your character's position, in order to avoid overshooting the specified dash distance.


This example demonstrates how a listener to this event can be helpful: 

using UnityEngine;

[RequireComponent(typeof(CharacterDash))]
public class Example : MonoBehaviour 
{
    [SerializeField] private CharacterDash characterDash;

    private void Awake()
    {
        characterDash = GetComponent<CharacterDash>();
    }

    private void Start()
    {
        characterDash.OnDashDistanceCovered += Dash_OnDashDistanceCovered;
    }

    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.LeftShift))
        {
            characterDash.Dash();
        }
    }

    private void Dash_OnDashDistanceCovered(object sender, CharacterDash.OnDashDistanceCoveredEventArgs e)
    {
        transform.position = e.targetDashPosition;
    }
}
