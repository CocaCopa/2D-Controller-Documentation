CharacterDash

SWITCH TO MANUAL

Dependencies

Required Components: Rigidbody2D, CharacterEnvironmentalQuery


Overview Note

Some public methods are integrated and managed by the HumanoidController. These methods are generally irrelevant for users relying on the full provided asset, but they are documented in case users wish to use the CharacterDash class separately.


Description

This class is designed to handle the behaviour of your character while dashing.


Public Properties
OnDashDistanceCovered Triggered when the dash action has been completed.
Public Properties
DashSpeed The speed of your character while dashing.
MaximumDashDistance The maximum distance your character can cover while dashing.
Public Methods - Direct Usage
DashForceStop Forces your character to stop the dash action.
Public Methods - Internal Integration
Dash Makes your character dash.
