CharacterDash.DashForceStop

Declaration

public void DashForceStop();


Description

Forces your character to stop the dash action.

This method is particularly useful when you need to immediately stop the dash of your character before the set distance is completed. For instance, if your character is dashing and gets killed, the dash action might not complete properly, causing your character to get stuck. In such cases, calling this function can prevent the issue by ensuring the dash is halted correctly.


In the example below, the dash will be forcibly stopped once the character has no more health points remaining. 

using UnityEngine;
using CharacterController2D.LocomotionSystem;
using CharacterController2D.CombatSystem;

[RequireComponent(typeof(CharacterDash), typeof(EntityHealth))]
public class Example : MonoBehaviour 
{
    private EntityHealth entityHealth;
    private CharacterDash characterDash;

    private void Awake()
    {
        entityHealth = GetComponent<EntityHealth>();
        characterDash = GetComponent<CharacterDash>();
    }

    private void Start()
    {
        entityHealth.OnEntityDeath += EntityHealth_OnEntityDeath;
    }

    private void EntityHealth_OnEntityDeath(object sender, System.EventArgs e)
    {
        characterDash.DashForceStop();
    }
}
