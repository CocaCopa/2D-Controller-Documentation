CharacterEnvironmentalQuery.LedgeGrabCheck

Declaration

public bool LedgeGrabCheck(ref bool exitLedgeGrab, out Vector3 fixedOffset);


Parameters
exitLedgeGrab Indicates that the ability to perform ledge grab is available again.
fixedOffset The position of the detected ledge.

Returns

True, if a ledge is detected in the game environment, otherwise false.


Description

Checks if the character is able to grab onto a ledge.

The function initiates a Physics2D.OverlapCircle cast and a Physics2D.OverlapBox cast. Successful detection of a ledge is confirmed if the circle cast returns true while the box cast returns false. In the event that both casts return true, it indicates the presence of a wall. The position of the circle is determined by the transform provided in the inspector of the component, while the position of the box lies on top of the circle.

Note the use of exitLedgeGrab in the example below. When you need your character to exit a ledge grab/climb state, set this flag to true. The LedgeGrabCheck function will then indicate when the character is ready to grab a ledge again. Once it returns false, signifying that a ledge is no longer detected, it will also reset the provided boolean back to false.


In this example, upon detecting a ledge, the character's rigidbody will be set to kinematic, and will log the position of the ledge in the console. 

using UnityEngine;

[RequireComponent(typeof(CharacterEnvironmentalQuery), typeof(Rigidbody2D))]
public class Example : Monobehaviour
{
    [SerializeField] private CapsuleCollider2D m_Collider;
    [SerializeField] private Rigidbody2D characterRb;

    private CharacterEnvironmentalQuery envQuery;
    private bool exitLedgeGrab = false;

    private void Awake()
    {
        characterRb = GetComponent⟨Rigidbody2D⟩();
        envQuery = GetComponent⟨CharacterEnvironmentalQuery⟩();
        envQuery.SetActiveCollider(m_Collider);
    }

    private void Update()
    {
        bool ledgeDetected = envQuery.LedgeGrabCheck(ref exitLedgeGrab, out Vector3 ledgePosition);
        
        if (!exitLedgeGrab && ledgeDetected)
        {
            Debug.Log("Ledge position: " + ledgePosition);
            characterRb.isKinematic = true;
        }

        // For convenience, if you wish to use this example, press the specified key below (Escape) to restore the rigidbody's state back to dynamic:
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            characterRb.isKinematic = false;
            exitLedgeGrab = true;
        }
    }
}
