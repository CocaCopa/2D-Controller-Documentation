CharacterEnvironmentalQuery.WallInFront

Declaration

public bool WallInFront();


Returns

True, if the character is standing in front of a wall, otherwise false.


Description

Checks if there is a wall in front of the character.

This function performs a Physics2D.CapsuleCast in the forward direction, taking into account the size and position of the activeCollider.


Here is an example of how you can use WallInFront in your script: 

using UnityEngine;

[RequireComponent(typeof(CharacterEnvironmentalQuery))]
public class Example : Monobehaviour
{
    [SerializeField] private CapsuleCollider2D m_Collider;

    private CharacterEnvironmentalQuery envQuery;

    private void Awake()
    {
        envQuery = GetComponent⟨CharacterEnvironmentalQuery⟩();
        envQuery.SetActiveCollider(m_Collider);
    }

    private void Update()
    {
        if (envQuery.WallInFront())
        {
            Debug.Log("The character is standing in front of a wall.");
        }
    }
}

Declaration

public bool WallInFront(float distance);


Parameters
distance Overrides the default cast distance.

Returns

True, if there is a wall found at the specified distance in front of the character, otherwise false.


Description

With this overload you can override the default cast distance.


Here is an example of how you can use this overload in your script: 

using UnityEngine;

[RequireComponent(typeof(CharacterEnvironmentalQuery))]
public class Example : Monobehaviour
{
    [SerializeField] private CapsuleCollider2D m_Collider;
    [SerializeField] private float distance = 4f;

    private CharacterEnvironmentalQuery envQuery;

    private void Awake()
    {
        envQuery = GetComponent⟨CharacterEnvironmentalQuery⟩();
        envQuery.SetActiveCollider(m_Collider);
    }

    private void Update()
    {
        if (envQuery.WallInFront(distance))
        {
            Debug.Log("There is a wall in front of the character at a maximum distance of " + distance + " Unity units away.");
        }
        else
        {
            Debug.Log("There are no colliders at the specified distance.");
        }
    }
}

Declaration

public bool WallInFront(RaycastHit2D hitInfo, float distance);


Parameters
hitInfo If true is returned, hitInfo will contain more information about where the closest collider was hit.
distance Overrides the default cast distance.

Returns

True, if there is a wall found at the specified distance in front of the character, otherwise false.


Description

Same as above, but it also returns information about the collider hit.


Here is an example of how you can use this overload in your script: 

using UnityEngine;

[RequireComponent(typeof(CharacterEnvironmentalQuery))]
public class Example : Monobehaviour
{
    [SerializeField] private CapsuleCollider2D m_Collider;
    [SerializeField] private float distance = 4f;

    private CharacterEnvironmentalQuery envQuery;

    private void Awake()
    {
        envQuery = GetComponent⟨CharacterEnvironmentalQuery⟩();
        envQuery.SetActiveCollider(m_Collider);
    }

    private void Update()
    {
        if (envQuery.WallInFront(out RaycastHit2D hitInfo, distance))
        {
            Debug.Log("Collider: " + hitInfo.transform.name + ", found in front of the character at a maximum distance of " + distance + " Unity units away.");
        }
        else
        {
            Debug.Log("There are no a colliders at the specified distance.");
        }
    }
}
