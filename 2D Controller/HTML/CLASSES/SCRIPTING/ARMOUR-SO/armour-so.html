ArmourSO

SWITCH TO MANUAL

Description

The ArmourSO is a Scriptable Object that stores the data of an armour object. You can manage the data manually using a custom script, but it's worth noting that within the EntityHealth class, this data is accessed to dynamically influence how damage taken is calculated for an entity.


ArmourName The name of the armour game object.
ArmourIcon The icon of the amour game object.
ArmourPoints The total armour points of the character/object.
ArmourEffectiveness The effectiveness of the armour. Lower values will increase the armour's strength, while higher values will decrease it.
