TryFloorSlide

Declaration

protected void TryFloorSlide(bool inputHold);


Parameters
inputHold Determines when the floor slide should be initiated

Description

Handles the logic for initiating and exiting floor sliding.

  • When the designated key for floor sliding is pressed, the controller initiates a sequence of checks to validate the feasibility of a floor slide. Initially, it examines the character's speed, ensuring it surpasses the 'minimumFloorSlideSpeed' parameter, which can be configured within the controller's inspector. If the character's speed falls short of this threshold, the function will not be executed.

  • Additionally, contingent on the settings of the 'CharacterFloorSlide' component, the character may experience a reduction in speed during the floor slide. In such instances, if the character's speed drops below the specified value, the floor slide will automatically halt.

  • Lastly, should the conditions for floor sliding cease, and the character finds itself beneath a wall, where their collider cannot fit under, the controller will persist in executing the floor slide until the character is no longer positioned beneath the obstructing wall.

This example demonstrates how you can use the 'TryFloorSlide' function within your custom script. 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        bool floorSlideKey = Input.GetKey(Keycode.DownArrow);
        TryFloorSlide(floorSlideKey);
    }
}
