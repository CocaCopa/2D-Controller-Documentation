LedgeClimb

Declaration

protected void LedgeClimb(bool canLedgeClimb = true);


Parameters
canLedgeClimb True, your character will automatically perform a ledge climb as soon as a ledge is detected

Description

Handles the logic for ledge climbing, allowing customization of conditions.

In alignment with the Ledge Grab configuration in the 'CharacterEnvironmentalQuery' component, the 'LedgeClimb();' method operates similarly to 'LedgeGrab();'. When a ledge is detected, your character gains the ability to climb it. You can trigger automatic ledge climbing by invoking 'LedgeClimb();' without passing any parameters, or by defining a custom condition, such as a player's key press. Much like the LedgeGrab method, you can fine-tune the behavior of the ledge climb mechanic by adjusting the settings of the 'CharacterLedgeGrab' component.

It's important to note that if your controller is set to utilize 'LedgeClimb();' without parameters, enabling automatic ledge climbing, this function will take precedence over 'LedgeGrab();'. If you intend to use both functions in your code, ensure that, at least 'LedgeClimb();' is configured to be triggered conditionally to prevent any unwanted behaviour.

This example incorporates the ledge climb mechanic, specifying that the player must run toward a wall for it to be activated: 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        float horizontalInput = Input.GetAxis("Horizontal");
        float moveDirection = transform.right.x;
        bool canLedgeClimb = horizontalInput * moveDirection > 0;

        LedgeClimb(canLedgeClimb);
    }
}

Here is an example on how you could use both ledge grab and ledge climb mechanics within your custom script: 

using UnityEngine;

public class Example : HumanoidController
{
    private bool canLedgeClimb = false;

    protected override void Update()
    {
        base.Update();

        if (Input.GetKeyDown(KeyCode.Space))
        {
            canLedgeClimb = true;
        }

        LedgeGrab();
        LedgeClimb(canLedgeClimb);

        if (!IsLedgeClimbing && canLedgeClimb)
        {
            canLedgeClimb = false;
        }
    }
}
