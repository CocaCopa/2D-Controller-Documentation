ChangeHorizontalVelocity

Declaration

protected void ChangeHorizontalVelocity(Vector2 moveDirection);


Parameters
moveDirection Desired horizontal direction to move towards

Description

Changes the character's horizontal velocity based on the provided direction.
  • This function manages the horizontal movement of the character, taking into account various factors such as the character's grounded or airborne state.

  • It automatically limits the character's movement if another action, such as 'dash,' is being performed.

  • Additionally, it sets the running state of the character by setting the value of the 'IsRunning' property accordingly

  • If a combat system is in place, it considers attack configurations and may restrict the character's movement accordingly.

This example implements the function to facilitate character movement. It's important to highlight that inheriting from the Humanoid controller is crucial for the proper functioning of this feature. 

using UnityEngine; 
 

public class Example : HumanoidController 

{
    protected override void Update()
    {
        base.Update();
        Vector2 moveDirection = Input.GetAxis("Horizontal");
        ChangeHorizontalVelocity(moveDirection);
    }
}
