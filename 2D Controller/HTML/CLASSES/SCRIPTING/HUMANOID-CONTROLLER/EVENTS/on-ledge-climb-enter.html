HumanoidController.OnLedgeClimbEnter

Declaration

public event EventHandler OnLedgeClimbEnter;


Description

Triggered when the character starts climbing a ledge.

When the HumanoidController detects a ledge, and if the condition passed to the LedgeClimb function is true, the OnLedgeClimbEnter event will be triggered.


To subscribe to this event from an external script, you must reference the equivalent HumanoidController class: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : MonoBehaviour 
{
    [SerializeField] private HumanoidController m_HumanoidController;
    
    private void Start()
    {
        m_HumanoidController.OnLedgeClimbEnter += HumanoidController_OnLedgeClimbEnter;
    }

    private void HumanoidController_OnLedgeClimbEnter(object sender, System.EventArgs e)
    {
        Debug.Log(m_HumanoidController.gameObject.name + ": initiated a ledge climb");

        // Code to execute when the event is triggered.
    }
}
