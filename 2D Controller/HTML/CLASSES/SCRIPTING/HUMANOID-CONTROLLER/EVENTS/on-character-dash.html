HumanoidController.OnCharacterDash

Declaration

public event EventHandler OnCharacterDash;


Description

Triggered when the character performs a dash.

Upon successful execution of the TryDashing function in your custom controller, your character performs a dash, triggering the OnCharacterDash event.


To subscribe to this event from an external script, you must reference the equivalent HumanoidController class: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : MonoBehaviour 
{
    [SerializeField] private HumanoidController m_HumanoidController;
    
    private void Start()
    {
        m_HumanoidController.OnCharacterDash += HumanoidController_OnCharacterDash;
    }

    private void HumanoidController_OnCharacterDash(object sender, System.EventArgs e)
    {
        Debug.Log(m_HumanoidController.gameObject.name + ": performed a dash");

        // Code to execute when the event is triggered.
    }
}
