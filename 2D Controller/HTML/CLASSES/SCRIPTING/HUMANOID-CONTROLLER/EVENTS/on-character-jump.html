HumanoidController.OnCharacterJump

Declaration

public event EventHandler OnCharacterJump;


Description

Triggered when the character performs a jump.

Upon successful execution of the TryJumping function in your custom controller, your character performs a jump, triggering the OnCharacterJump event in the HumanoidController.


To subscribe to this event from an external script, you must reference the equivalent HumanoidController class: 

using UnityEngine;

public class Example : Monobehaviour 
{
    [SerializeField] private HumanoidController m_HumanoidController;
    
    private void Awake()
    {
        m_HumanoidController.OnCharacterJump += HumanoidController_OnCharacterJump;

        // Exercise caution when subscribing to events. If you plan to initialize 'm_HumanoidController' within the 'Awake' function,
        // ensure that the event subscription takes place in the 'Start' method instead.
    }

    private void HumanoidController_OnCharacterJump(object sender, System.EventArgs e)
    {
        Debug.Log(m_HumanoidController.gameObject.name + ": performed a jump");
        
        // Code to execute when the event is triggered.
    }
}
