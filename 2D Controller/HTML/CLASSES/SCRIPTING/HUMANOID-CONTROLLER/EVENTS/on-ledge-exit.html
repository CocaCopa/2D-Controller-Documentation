OnLedgeExit

Declaration

public event EventHandler OnLedgeExit;


Description

Triggered when the character exits the ledge grab or ledge climb state.

When your character ceases to grab a ledge or completes a ledge climb, the HumanoidController will trigger the OnLedgeExit event.


To subscribe to this event from an external script, you must reference the equivalent HumanoidController class: 

using UnityEngine;

public class Example : Monobehaviour 
{
    [SerializeField] private HumanoidController m_HumanoidController;
    
    private void Awake()
    {
        m_HumanoidController.OnLedgeClimbEnter += HumanoidController_OnLedgeExit;

        // Exercise caution when subscribing to events. If you plan to initialize 'm_HumanoidController' within the 'Awake' function,
        // ensure that event subscriptions take place in the 'Start' method.
    }

    private void HumanoidController_OnLedgeExit(object sender, System.EventArgs e)
    {
        Debug.Log(m_HumanoidController.gameObject.name + ": is no longer performing ledge grab nor ledge climb");

        // Code to execute when the event is triggered.
    }
}
