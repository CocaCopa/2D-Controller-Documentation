HumanoidController.OnLedgeGrabEnter

Declaration

public event EventHandler OnLedgeGrabEnter;


Description

Triggered when the character enters the ledge grab state.

When the HumanoidController detects a ledge, and if the condition passed to the LedgeGrab function is true, the OnLedgeGrabEnter event will be triggered.


To subscribe to this event from an external script, you must reference the equivalent HumanoidController class: 

using UnityEngine;

public class Example : MonoBehaviour 
{
    [SerializeField] private HumanoidController m_HumanoidController;
    
    private void Start()
    {
        m_HumanoidController.OnLedgeGrabEnter += HumanoidController_OnLedgeGrabEnter;
    }

    private void HumanoidController_OnLedgeGrabEnter(object sender, System.EventArgs e)
    {
        Debug.Log(m_HumanoidController.gameObject.name + ": grabbed a ledge");

        // Code to execute when the event is triggered.
    }
}
