HumanoidController.TryDashing

Declaration

protected void TryDashing();


Description

Attempts to make the character perform a dash based on certain conditions.

When the designated key for dashing is pressed, the controller will endeavor to make your character dash. You have control over when the dash is initiated through the 'dashCooldown' and 'minimumDashDistance' properties in the inspector of the controller. Tweak these properties to tailor the dash experience to your preferences. For a deeper level of customization, dive into the settings of the 'CharacterDash' component to fine-tune the intricacies of the dash mechanic.

Here is a basic example on how you can implement the dash mechanic into your custom controller script: 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        if (Input.GetKeyDown(KeyCode.LeftShit))
        {
            TryDashing();
        }
    }
}
