HumanoidController.ChangeHorizontalVelocity

Declaration

protected void ChangeHorizontalVelocity(Vector2 moveDirection);


Parameters
moveDirection Desired horizontal direction to move towards

Description

Changes the character's horizontal velocity based on the provided direction.

This function adjusts the character's horizontal velocity based on the provided direction. It manages the character's horizontal movement, considering factors such as whether the character is grounded or airborne. Additionally, it automatically restricts the character's movement if other actions, such as dashing, are occurring (refer to HumanoidController.TryDashing). In cases involving a combat system, the function also accounts for attack configurations and may impose movement limitations accordingly (refer to CharacterCombat).

Note: This function alone will not alter the orientation of the character. To modify the character's orientation while they move, also utilize the FlipCharacter function.


This example implements the function to facilitate character movement: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();
        Vector2 moveDirection = Input.GetAxis("Horizontal");
        ChangeHorizontalVelocity(moveDirection);
    }
}
