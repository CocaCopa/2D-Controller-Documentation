LedgeGrab

Declaration

protected void LedgeGrab(bool canLedgeGrab = true);


Parameters
canLedgeGrab True, your character will automatically perform a ledge grab as soon as a ledge is detected

Description

Handles the logic for ledge grabbing, allowing customization of conditions.

In accordance with the Ledge Grab configuration in the 'CharacterEnvironmentalQuery' component, when a ledge is detected, your character will be able to grab it. You have the option to initiate a grab either automatically, by calling 'LedgeGrab();' without passing any parameters, or by specifying a custom condition, such as a key press by the player. In the inspector of the controller, you can specify the maximum time that the character is able to hold a ledge, before being forced to cancel it, by configuring the 'maxLedgeGrabTime' value. Further customization of the ledge grab mechanic can be achieved by fine-tuning the settings of the 'CharacterLedgeGrab' component.

This example incorporates the ledge grab mechanic, specifying that the player must run toward a wall for it to be activated: 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        float horizontalInput = Input.GetAxis("Horizontal");
        float moveDirection = transform.right.x;
        bool canLedgeGrab = horizontalInput * moveDirection > 0;

        LedgeGrab(canLedgeGrab);
    }
}
