FlipCharacter

Declaration

protected void FlipCharacter(float directionX);


Parameters
directionX The direction the character should face.

Description

Reorients the character to face the specified horizontal direction.

This function adjusts the character's euler angles on the Y-axis based on the provided direction, ensuring that the character faces the specified orientation.


Here is an example of how you can use this function within your custom controller script:

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        float horizontalInput = Input.GetAxis("Horizontal");
        FlipCharacter(horizontalInput);
    }
}
