HumanoidController.WallSlide

Declaration

protected void WallSlide(bool canWallSlide = true);


Description

Handles the logic for wall sliding, based on the given condition.


Parameters
canWallSlide Condition for when the wall slide should be initiated, when near a wall

When your character is positioned on a wall, its ability to slide is determined by the conditions specified in the function. If the function is called without parameters, the character will automatically initiate a slide on that wall. The speed at which your character accelerates during the slide can be controlled by configuring the settings of the 'CharacterSlide' component.

In this example, the player needs to maintain a continuous run toward the wall for the function to be executed: 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        float horizontalInput = Input.GetAxis("Horizontal");
        float moveDirection = transform.right.x;
        bool canWallSlide = horizontalInput * moveDirection > 0;

        WallSlide(canWallSlide);
    }
}
