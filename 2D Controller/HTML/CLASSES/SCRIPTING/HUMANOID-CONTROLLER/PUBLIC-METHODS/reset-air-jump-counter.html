HumanoidController.ResetAirJumpCounter

Declaration

public void ResetAirJumpCounter();


Description

Resets the air jump counter back to its default value.

This method is particularly useful when implementing power-ups that affect the character's jump capabilities. For instance, if your character has exhausted all air jumps and then collects a power-up, the jump counter will reset.


Here's how you can implement the power-up. Create a new GameObject, then attach a Collider2D component and set it as trigger. Finally, attach the script provided below. When you enter play mode, if your character collects the power-up game object, their air jump counter will reset. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D collider)
    {
        if (collider.TryGetComponent<HumanoidController>(out var controller))
        {
            controller.ResetAirJumpCounter();
        }
    }
}
