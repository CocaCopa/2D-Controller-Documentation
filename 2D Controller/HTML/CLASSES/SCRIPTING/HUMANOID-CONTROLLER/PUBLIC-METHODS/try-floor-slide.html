HumanoidController.TryFloorSlide

Declaration

protected void TryFloorSlide(bool inputHold);


Parameters
inputHold Determines when the floor slide should be initiated

Description

Handles the logic for initiating and exiting floor sliding.

Upon initiating a floor-slide, your character's colliders will be toggled, making the Horizontal collider enabled and the Vertical collider disabled.

  • When the designated key for floor sliding is pressed, the controller initiates a sequence of checks to validate the feasibility of a floor slide. Initially, it examines the character's speed, ensuring it surpasses the specified threshold. If the character's speed falls short of the threshold, the function will not execute its logic.

  • Additionally, contingent on the settings of the 'CharacterSlide' component, the character may experience a reduction in speed during the floor slide. In such instances, if the character's speed drops below the floor slide threshold, the function will automatically halt its execution.

  • Lastly, should the conditions for floor sliding cease, and the character finds themselves beneath a wall, where their collider cannot fit under, the controller will persist in executing its logic, until the character is no longer positioned beneath the obstructing wall.

This example demonstrates how you can call the function within your custom controller script: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        bool floorSlideKey = Input.GetKey(Keycode.DownArrow);
        TryFloorSlide(floorSlideKey);
    }
}
