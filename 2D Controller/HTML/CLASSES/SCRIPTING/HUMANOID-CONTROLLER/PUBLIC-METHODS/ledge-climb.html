HumanoidController.LedgeClimb

Declaration

protected void LedgeClimb(bool canLedgeClimb = true);


Parameters
canLedgeClimb True, your character will automatically perform a ledge climb as soon as a ledge is detected

Description

Handles the logic for ledge climbing, allowing customization of conditions.

In alignment with the Ledge Grab configuration in the 'CharacterEnvironmentalQuery' component, the 'LedgeClimb();' method operates similarly to 'LedgeGrab();'. When a ledge is detected, your character gains the ability to climb it. You can trigger automatic ledge climbing by invoking the function without passing any parameters, or by defining a custom condition, such as a player's key press. Much like the ledge grab mechanic, you can fine-tune the behaviour of the ledge climb mechanic by adjusting the settings of the 'CharacterLedgeGrab' component.

It's important to note that if your controller is set to utilize 'LedgeClimb();' without parameters, enabling automatic ledge climbing, this function will take precedence over 'LedgeGrab();'. If you intend to use both mechanics, ensure that, at least 'LedgeClimb();' is configured to be triggered conditionally in order to prevent any unwanted behaviour.

This example incorporates the ledge climb mechanic, specifying that a certain key must be held for it to be activated: 

using UnityEngine;

public class Example : HumanoidController
{
    protected override void Update()
    {
        base.Update();

        bool ledgeClimbKey = Input.GetKey(KeyCode.F);
        LedgeClimb(ledgeClimbKey);
    }
}

Illustrated below is a demonstration of integrating both ledge grab and ledge climb mechanics into your custom controller script. Upon detecting a ledge, your character will promptly initiate a grab, and if the 'Space' key is pressed during the grab, it will seamlessly proceed to climb the ledge. 

using UnityEngine;

public class Example : HumanoidController
{
    private bool canLedgeClimb = false;

    protected override void Update()
    {
        base.Update();

        if (IsLedgeGrabbing && Input.GetKeyDown(KeyCode.Space))
        {
            canLedgeClimb = true;
        }

        LedgeGrab();
        LedgeClimb(canLedgeClimb);

        if (!IsLedgeClimbing && canLedgeClimb)
        {
            canLedgeClimb = false;
        }
    }
}
