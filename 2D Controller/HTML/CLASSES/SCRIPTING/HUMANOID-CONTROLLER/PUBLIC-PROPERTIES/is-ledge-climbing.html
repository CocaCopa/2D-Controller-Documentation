HumanoidController.IsLedgeClimbing

Declaration

public bool IsLedgeClimbing { get; }


Description

Indicates whether the character is currently climbing a ledge.

This property indicates whether the character is climbing ledge. It returns true if LedgeDetected is true and the LedgeClimb function of the HumanoidController class is referenced in your custom controller script.

If you need to access this property through a class that inherits from HumanoidController, you can do so directly. Alternatively, if you intend to access it from an external script, you need to reference the corresponding controller instance and access it accordingly.


In this example, we access the propery directly: 

using UnityEngine;

public class Example : HumanoidController 
{
    private void Update()
    {
        Debug.Log(IsRunning);
    }
}

In this example we access the property through an external script: 

using UnityEngine;

public class Example : Monobehaviour
{
    [SerializeField] private HumanoidController m_HumanoidController;

    private void Update()
    {
        Debug.Log(m_HumanoidController.IsRunning);
    }
}
