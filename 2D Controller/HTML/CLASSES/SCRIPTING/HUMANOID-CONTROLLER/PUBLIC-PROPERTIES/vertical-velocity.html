HumanoidController.VerticalVelocity

Declaration

public float VerticalVelocity { get; }


Description

Current velocity of the character on the Y-axis.

This property provides information about the character's speed in the vertical direction. It is specifically obtained from the Y-axis velocity of the character's Rigidbody2D.

If you need to access this property through a class that inherits from HumanoidController, you can do so directly. Alternatively, if you intend to access it from an external script, you need to reference the corresponding controller instance and access it accordingly.


In this example, we access the propery directly: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : HumanoidController 
{
    private void Update()
    {
        Debug.Log(VerticalVelocity);
    }
}

In this example we access the property through an external script: 

using UnityEngine;
using CharacterController2D.Core;

public class Example : MonoBehaviour
{
    [SerializeField] private HumanoidController m_HumanoidController;

    private void Update()
    {
        Debug.Log(m_HumanoidController.VerticalVelocity);
    }
}
