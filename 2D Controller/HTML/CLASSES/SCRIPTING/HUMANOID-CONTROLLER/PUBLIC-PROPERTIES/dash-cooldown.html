HumanoidController.DashCooldown

Declaration

public float DashCooldown { get; set; }


Description

The set cooldown of the dash mechaninc in seconds.

You can use this property to read the current cooldown of the dash, or change it if certain conditions are met.


In this example, we access the property directly, reading the cooldown of the dash: 

using UnityEngine;

public class Example : HumanoidController 
{
    private void Update()
    {
        Debug.Log(DashCooldown);
    }
}

In this example, the cooldown of the dash will be reset whenever the character collects a power-up. Create a gameObject of any shape and attach a collider to it, ensuring it's set as a trigger. Finally, attach the provided class below to the object. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.TryGetComponent<HumanoidController>(out var controller))
        {
            controller.DashCooldown = 0f;
        }
    }
}
