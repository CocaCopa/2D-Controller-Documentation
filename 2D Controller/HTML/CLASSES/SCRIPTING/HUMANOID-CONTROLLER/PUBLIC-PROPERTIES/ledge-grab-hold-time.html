HumanoidController.LedgeGrabHoldTime

Declaration

public float LedgeGrabHoldTime { get; }


Description

The set time that the character is able to stay idle while grabbing a ledge, before they are forced to exit 'LedgeGrab' state.


In this example, we access the propery directly: 

using UnityEngine;

public class Example : HumanoidController 
{
    private void Update()
    {
        Debug.Log(LedgeGrabHoldTime);
    }
}

In this example we access the property through an external script: 

using UnityEngine;

public class Example : Monobehaviour
{
    [SerializeField] private HumanoidController m_HumanoidController;

    private void Update()
    {
        Debug.Log(m_HumanoidController.LedgeGrabHoldTime);
    }
}
