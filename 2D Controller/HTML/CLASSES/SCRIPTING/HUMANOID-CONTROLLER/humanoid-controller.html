Humanoid Controller

Description

The HumanoidController class serves as an abstract base class for implementing a 2D character controller. It provides a comprehensive set of functionalities for handling:
  • Ground Movement
  • Aerial Movement
  • Jumping
  • Dashing
  • Ledge Grabbing
  • Ledge Climbing
  • Floor Sliding
  • Wall Sliding
You can inherit from this class to create custom humanoid characters with dynamic and responsive behaviors.
