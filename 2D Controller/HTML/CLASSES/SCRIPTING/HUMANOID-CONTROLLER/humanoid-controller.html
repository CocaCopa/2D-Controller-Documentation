Humanoid Controller

Description

The HumanoidController class serves as an abstract base class for implementing a 2D character controller. It provides a comprehensive set of functionalities for handling:
  • Ground/Aerial Movement
  • Jumping
  • Dashing
  • Ledge Grabbing
  • Ledge Climbing
  • Floor Sliding
  • Wall Sliding
You can inherit from this class to create custom humanoid characters, controlled by a player and/or an AI, with dynamic and responsive behaviors.


Events
OnCharacterJump
 Triggered when the character performs a jump
OnCharacterDash
 Triggered when the character performs a dash.
OnLedgeGrabEnter
 Triggered when the character enters the ledge grab state.
OnLedgeClimbEnter
 Triggered when the character starts climbing a ledge.
OnLedgeExit
 Triggered when the character exits the ledge grab or ledge climb state.

 Current velocity of the character on the X-axis.

 Indicates whether the character is grounded.

Public Methods
