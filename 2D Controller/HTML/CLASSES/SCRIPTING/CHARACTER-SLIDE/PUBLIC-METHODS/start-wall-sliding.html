CharacterSlide.StartWallSliding

Declaration

public float StartWallSliding();


Description

Controls the behaviour of the character for as long as the character is wall sliding.

When called, this function will reset the velocity of the character's Rigidbody2D and set the gravity scale to 0. It implements its own gravity logic, allowing you to control the 'hardness' of the gravitational pull by configuring the 'wallSlideAcceleration' value through the controller's inspector.


This is an example of how you can call StartWallSliding in your script. CharacterEnvironmentalQuery is also being used, to help with the wall detection, as well as HumanoidController to identify when the character is grounded:

        }
    }
