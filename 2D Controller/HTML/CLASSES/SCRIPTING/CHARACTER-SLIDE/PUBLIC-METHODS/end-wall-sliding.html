CharacterSlide.EndWallSliding

Declaration

public float EndWallSliding();


Description

Resets certain values when the character exits the wall slide.

This function should be called after calling StartWallSliding to reset the character's state back to default.


This is an example of how you can call EndWallSliding in your script. CharacterEnvironmentalQuery is also being used, to help with the wall detection, as well as HumanoidController to identify when the character is grounded: 

using UnityEngine;

[RequireComponent(typeof(CharacterSlide), typeof(CharacterEnvironmentalQuery))]
public class Example : HumanoidController 
{

        }
        else

        }
    }
}
