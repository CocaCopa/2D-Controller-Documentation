CharacterSlide.FloorSlide

Declaration

public float FloorSlide(float minimumSpeed);


Parameters
minimumSpeed Character speed will stop being reduced when it reaches the specified value.

Description

Manages the character's behaviour, allowing for a floor slide during the ongoing function call.

Once this function gets called, your character will start sliding on the floor, decreasing the speed of their Rigidbody2D by the specified 'floorSlideDesceleration' value, set in the inspector of the component.

Note that the function only executes the raw part of the mechanic. If additional behaviour, such as continuing the slide when the character is under a ceiling, is required, you must implement it independently.


This is an example of how you can call FloorSlide in your script: 

using UnityEngine;

[RequireComponent(typeof(CharacterSlide))]
public class Example : MonoBehaviour 
{
    [SerializeField] private CharacterSlide characterSlide;
    [SerializeField] private float minimumSlideSpeed;

    private void Awake()
    {
        characterSlide = GetComponent<CharacterSlide>();
    }

    private void Update()
    {
        if (Input.GetKey(Keycode.DownArrow))
        {
            characterSlide.FloorSlide(minimumSlideSpeed);
        }
    }
}
