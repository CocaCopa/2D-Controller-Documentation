CharacterSlide

SWITCH TO MANUAL

Dependencies

Required Component: Rigidbody2D


Overview Note

All public methods are integrated and managed by the HumanoidController. These methods are generally irrelevant for users relying on the full provided asset, but they are documented in case users wish to use the CharacterSlide class separately.


Description

This class is responsible for controlling your character when they are sliding either on a floor or on a wall.


Public Properties
FloorSlideDeceleration The speed at which the character decelerates during a floor slide.
WallSlideAcceleration The speed at which the character accelerates during a wall slide.
Public Methods - Internal Integration
FloorSlide Controls the behaviour of the character for as long as the character is floor sliding.
StartWallSliding Controls the behaviour of the character for as long as the character is wall sliding.
EndWallSliding Resets certain values when the character exits the wall slide.
