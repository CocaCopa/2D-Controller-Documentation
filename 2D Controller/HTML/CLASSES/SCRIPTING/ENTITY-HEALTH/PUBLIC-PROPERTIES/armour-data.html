EntityHealth.ArmourData

Declaration

public ArmourSO ArmourData { get; set; }


Description

The armour data (if any) of the entity.

By creating an ArmourSO scriptable object, you can assign it to the EntityHealth component and give your character armour stats.


Here is an example of how you could use the property. Create a new game object with a Collider2D of your choice and set it as a trigger. Add to that newly created game object the tag "ArmourPowerUp". Finally, attach this script to your character. When the character collides with that game object, the armour data of the EntityHealth script will change to the one defined by the following script. 

using UnityEngine;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    [SerializeField] private ArmourSO empoweredArmour;

    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
    }
    
    private void OnTriggerEnter2D(Collider2D collider)
    {
        if (collider.CompareTag("ArmourPowerUp"))
        {
            healthScript.ArmourData = empoweredArmour;
        }
    }
}
