EntityHealth.RegenTriggerTime

Declaration

public float RegenTriggerTime { get; set; }


Description

Time in seconds before health regeneration triggers.

This property dictates the delay between the occurrence of health recovery. Adjusting this property allows for precise control over the timing and frequency of health restoration actions within the gameplay dynamics.


In this example, if the character picks up a health power-up, the time needed for the health regeneration of the character to take effect is halved.
Assign the "Player" tag to your character object and attach both a Collider2D and Rigidbody2D component to it. Create another game object with any shape and a Collider2D component, setting it as a trigger. Lastly, attach the provided class to this object. 

using UnityEngine;
using CharacterController2D.CombatSystem;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.CompareTag("Player"))
        {
            EntityHealth healthScript = col.GetComponent<EntityHealth>();
            healthScript.RegenTriggerTime /= 2f;
        }
    }
}
