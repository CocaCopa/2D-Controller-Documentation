EntityHealth.CanRegenHealth

Declaration

public bool CanRegenHealth { get; set; }


Description

Indicates whether your entity is able to regen missing health points.

This property determines whether the entity has the ability to regenerate health over time during gameplay. It can be toggled dynamically during gameplay to either prevent the character from regenerating missing health points or grant them the ability to do so, regardless of their current health status.


In this example, if the character picks up a health power-up, they will be able to regen health.
Create a game object of any shape with a Collider2D of your choice and set it as a trigger. Finally attach the provided class to the object. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.TryGetComponent<HumanoidController>(out var controller))
        {
            EntityHealth healthScript = controller.GetComponent<EntityHealth>();
            healthScript.CanRegenHealth = true;
        }
    }
}
