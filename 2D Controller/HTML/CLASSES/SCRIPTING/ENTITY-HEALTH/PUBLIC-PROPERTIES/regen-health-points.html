EntityHealth.RegenHealthPoints

Declaration

public float RegenHealthPoints { get; set; }


Description

The amount of health points the entity should regenerate, once the health regeneration effect has been triggered.

This property determines the amount of health restored to the entity over a specified duration, providing a means for replenishing health during gameplay. Adjusting this property allows for fine-tuning the effectiveness of health regeneration mechanics, balancing the pace and sustainability of healing within the game environment.


In this example, if the character picks up a health power-up, the amount of health points gained during health regeneration will be doubled.
Assign the "Player" tag to your character object and attach both a Collider2D and Rigidbody2D component to it. Create another game object with any shape and a Collider2D component, setting it as a trigger. Lastly, attach the provided class to this object. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.CompareTag("Player"))
        {
            EntityHealth healthScript = col.GetComponent<EntityHealth>();
            healthScript.RegenHealthPoints *= 2f;
        }
    }
}
