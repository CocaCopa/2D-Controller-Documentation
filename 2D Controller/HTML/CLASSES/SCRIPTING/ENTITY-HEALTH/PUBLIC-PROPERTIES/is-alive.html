EntityHealth.IsAlive

Declaration

public bool IsAlive { get; }


Description

Indicates whether the entity is alive.


Here's an example demonstrating the utility of this property: Assuming the following class represents a player controller, we can leverage the property's value to constrain the player's controls when their character is dead. 

using UnityEngine;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
    }

    private void Update()
    {
        if (healthScript.IsAlive == false)
        {
            return;
        }

        // Your character controller code.
    }
}
