EntityHealth.CurrentHealthPoints

Declaration

public float CurrentHealthPoints { get; set; }


Description

The current health points of the entity.

This property dynamically tracks the entity's current health status during gameplay, reflecting any changes due to damage taken, healing effects, or other interactions. Note that setting the current health points of the character to a value exceeding the EntityHealth.MaxHealthPoints will result in the property being set to that maximum instead.


In this example, if the character picks up a health power-up, their current health points will be doubled.
Assign the "Player" tag to your character object and attach both a Collider2D and Rigidbody2D component to it. Create another game object with any shape and a Collider2D component, setting it as a trigger. Lastly, attach the provided class to this object. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.CompareTag("Player"))
        {
            EntityHealth healthScript = col.GetComponent<EntityHealth>();
            healthScript.CurrentHealthPoints *= 2f;
        }
    }
}
