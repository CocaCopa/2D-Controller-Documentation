EntityHealth.MaxHealthPoints

Declaration

public float MaxHealthPoints { get; set; }


Description

Maximum health points of the entity.

This property represents the maximum health points available to an entity within your game. By setting this property, you define the upper limit for the health points of the entity. It allows for easy access and modification of the maximum health value, which can be crucial for gameplay mechanics such as health upgrades, damage scaling, and more.


In this example, if the character picks up a health power-up, their maximum health points will be doubled.
Assign the "Player" tag to your character object and attach both a Collider2D and Rigidbody2D component to it. Create another game object with any shape and a Collider2D component, setting it as a trigger. Lastly, attach the provided class to this object. 

using UnityEngine;
using CharacterController2D.CombatSystem;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.CompareTag("Player"))
        {
            EntityHealth healthScript = col.GetComponent<EntityHealth>();
            healthScript.MaxHealthPoints *= 2f;
        }
    }
}
