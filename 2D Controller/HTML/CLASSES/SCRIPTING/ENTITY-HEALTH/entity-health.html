EntityHealth

SWITCH TO MANUAL

Description

This class manages health-related aspects for entities in the game, including tracking health points, handling damage events, and managing health regeneration. It provides extensible functionality to implement specialized health behaviours and integrate additional features as needed.

Note: This class implements the IDamageable interface which can be found within the asset.


Events
OnEntityAlive This event is triggered when the entity is spawned into the game world.
OnEntityDeath This event is triggered when an entity dies or is destroyed within the game world.
OnTakeDamage This event is triggered when an entity receives damage from an external source.
Public Properties
IsAlive Indicates whether the entity is alive.
ArmourData The armour data (if any) of the entity.
MaxHealthPoints Maximum health points of the entity.
CurrentHealthPoints The current health points of the entity.
CanRegenHealth Indicates whether your entity is able to regen missing health points.
InterruptHealthRegenWhenDamaged Indicates whether your entity should stop regenerating health, when they take damage.
RegenHealthPoints The amount of health points the entity should regenerate, once the health regeneration effect has been triggered.
RegenTriggerTime Time in seconds before health regeneration triggers.
RegenTriggerTimer Triggers health regeneration when this value reaches 0.
IsRegeneratingHealth Indicates whether the entity is regenerating health points.
Public Methods
ResetRegenerationTimer Resets the health regeneration timer.
TakeDamage Inflicts damage on the entity.
EntityRespawned Resets entity health values upon respawn.
StartHealthRegeneration Starts health regeneration.
