EntityHealth.TakeDamage

Declaration

public void TakeDamage(float amount);


Parameters
amount The amount of damage the entity will receive.

Description

Inflicts damage on the entity.

When called, this method deducts a specified amount of health points from the entity's current health pool, simulating the effects of damage inflicted by various game mechanics such as character attacks.


In the example below, the entity will take damage when the 'Escape' key is pressed. 

using UnityEngine;
using CharacterController2D.CombatSystem;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
    }
        
    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            healthScript.TakeDamage(25f);
        }
    }
}
