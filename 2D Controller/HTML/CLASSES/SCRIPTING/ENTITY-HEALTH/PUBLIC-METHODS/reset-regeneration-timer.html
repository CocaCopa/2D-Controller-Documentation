EntityHealth.ResetRegenerationTimer

Declaration

public void ResetRegenerationTimer();


Description

Resets the health regeneration timer.

Be cautious when dealing damage to entities if you intend to use the health regeneration effect. When an entity's health dips below 100%, its health regeneration timer remains unaffected by incoming damage. Often, you'll want the regeneration timer to reset when damage occurs. This function accomplishes just that. Remember to invoke it whenever your entity takes damage.


In the example below, the entity will take damage if the 'Escape' key is pressed, and the health regeneration timer will reset. 

using UnityEngine;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
        healthScript.OnTakeDamage += Health_OnTakeDamage;
    }
        
    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            healthScript.TakeDamage(25f);
        }
    }

    private void Health_OnTakeDamage(object sender, EntityHealth.OnTakeDamageEventArgs e) {
        healthScript.ResetRegenerationTimer();
    }
}
