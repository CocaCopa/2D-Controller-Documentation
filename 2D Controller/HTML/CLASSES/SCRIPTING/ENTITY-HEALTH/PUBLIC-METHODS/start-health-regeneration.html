EntityHealth.StartHealthRegeneration

Declaration

public void StartHealthRegeneration(bool immediate);


Parameters
immediate Determines if the regeneration should happen immediately or wait for the 'RegenTriggerTimer' value first.

Description

Starts health regeneration.

This method initiates the health regeneration process for the entity. When called, it triggers the activation of mechanisms that gradually restore the entity's health points over time.


In this example, if the character picks up a health power-up, the health regeneration effect will be immediately triggered.
Give your character object the tag "Player". Create a game object of any shape with a Collider2D of your choice and set it as a trigger. Finally attach the provided class to the object.
For convinience, once the character object collects the power-up, they will also receive damage. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private void OnTriggerEnter2D(Collider2D col)
    {
        if (col.CompareTag("Player"))
        {
            EntityHealth healthScript = col.GetComponent<EntityHealth>();
            healthScript.TakeDamage(healthScript.CurrentHealthPoints / 2f);
            healthScript.StartHealthRegeneration(true);
        }
    }
}
