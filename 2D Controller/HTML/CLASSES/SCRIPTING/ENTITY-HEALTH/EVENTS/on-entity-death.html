EntityHealth.OnEntityDeath

Declaration

public event EventHandler OnEntityDeath;


Description

This event is triggered when an entity dies or is destroyed within the game world.

"This event is triggered when an entity meets its demise or is destroyed within the game world. This event serves as a notification mechanism for signaling the occurrence of an entity's death to other game components or systems. You can subscribe to this event to implement various post-death behaviors or actions, such as playing death animations, triggering particle effects, removing the entity from the scene, updating scoreboards, or initiating game over conditions.


To use the example below, attach the provided script to a 2D game object of your choice. When you press the 'Escape' key on your keyboard, a message will be logged in the console, indicating that the entity has died. 

using UnityEngine;
using CharacterController2D.CombatSystem;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
        healthScript.MaxHealthPoints = 100f;
        healthScript.CurrentHealthPoints = 100f;

        healthScript.OnEntityDeath += HealthScript_OnEntityDeath;
    }

    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            healthScript.TakeDamage(100f);
        }
    }

    private void HealthScript_OnEntityDeath(object sender, EventArgs e)
    {
        Debug.Log("The entity died.");
    }
}
