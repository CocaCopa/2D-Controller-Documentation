EntityHealth.OnEntityAlive

Declaration

public event EventHandler OnEntityAlive;


Description

This event is triggered when the entity is spawned into the game world.

This event is triggered when the entity is spawned into the game world. This event serves as a notification mechanism to inform other game components or systems about the entity's existence and readiness for interaction within the game environment. You can subscribe to this event to perform initialization tasks or set up initial conditions for the entity upon its creation, such as positioning the entity, initializing properties, activating behaviors, or spawning associated objects.


To use the example below, attach the provided script to a 2D game object of your choice. When you press the 'Escape' key on your keyboard, a message will be logged in the console, indicating that the entity has respawned. 

using UnityEngine;

[RequireComponent(typeof(EntityHealth))]
public class Example : MonoBehaviour
{
    private EntityHealth healthScript;

    private void Awake()
    {
        healthScript = GetComponent<EntityHealth>();
        healthScript.MaxHealthPoints = 100f;
        healthScript.CurrentHealthPoints = 100f;

        healthScript.OnEntityAlive += HealthScript_OnEntityAlive;
    }

    private void Start()
    {
        healthScript.TakeDamage(100f);
    }

    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            healthScript.Alive();
        }
    }

    private void HealthScript_OnEntityAlive(object sender, EventArgs e)
    {
        Debug.Log("The entity respawned at full health.");
    }  
}
