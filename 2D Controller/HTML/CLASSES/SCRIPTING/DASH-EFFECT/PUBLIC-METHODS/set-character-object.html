DashEffect.SetCharacterObject

Declaration

public void SetCharacterObject(GameObject character);


Parameters
character The game object representing the character.

Description

Sets the character object to track for the dash effect.

This method allows you to specify the character object that the dash effect should follow during the dash sequence.


Here is an example of how to utilize this method. It's important to invoke this method after instantiating the after-image prefab. For demonstration purposes the after-image object will be instantiated with the press of the 'Escape' button. 

using UnityEngine;

public class Example : MonoBehaviour
{
    [SerializeField] GameObject afterImageObject;

    private void Update()
    {
        if (Input.GetKeyDown(KeyCode.Escape))
        {
            GameObject afterImage = Instantiate(afterImageObject, Vector3.zero, Quaternion.identity);
            afterImage.TryGetComponent<DashEffect>(out var dashEffect);
            dashEffect?.SetCharacterObject(gameObject);
        }
    }
}
