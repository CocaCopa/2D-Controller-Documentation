CombatSystemProjectile

Manual

Dependencies

Required Components: Collider2D, Rigidbody2D


Description

This class is designed for game objects meant to act as projectiles. It controls how these projectiles move and deal damage to other objects. To be damaged by these projectiles, an object must implement the IDamageable interface.

It is worth noting that this class pairs well with the CharacterCombat class.


Public Properties
InitialVelocity The initial velocity of the projectile when it's launched.
DamageLayers Indicates which layers can be damaged.
DamageAmount Indicates the amount of damage the projectile will deal upon collision with a damageable object.
Public Methods
DealDamageOnContact Deals damage to the first collider that enters the specified hit box.
ContinuousAreaOfEffectDamage Deals damage instantly to any new colliders that enter the specified hitbox and then deals damage again, to all targets, every 'x' amount of seconds.
ExplodeOnContact Deals damage to all colliders inside the specified hitbox.
