ComatSystemProjectile

Dependencies

Required Components: CharacterEnvironmentalQuery, Rigidbody2D


Description

Description of the combat system projectile class goes here.

**IMPORTANT TO ADD**
For an object to take damage from a projectile, it needs to implement the IDamageable interface.


Public Properties
InitialVelocity The initial velocity of the projectile when it's launched.
DamageLayers Indicates which layers can be damaged.
DamageAmount Indicates the amount of damage the projectile will deal upon collision with a damageable object.
HitboxShape Indicates the shape of the projectile's hitbox. It can be a cirle or a box.
Public Methods
DealDamageOnContact Deals damage to the first collider that enters the specified hit box.
ContinuousAreaOfEffectDamage Deals damage instantly to any new colliders that enter the specified hitbox and then deals damage again, to all targets, every 'x' amount of seconds.
ExplodeOnContact Deals damage to all colliders inside the specified hitbox and then destroys the projectile.
