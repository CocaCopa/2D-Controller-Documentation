CombatSystemProjectile.DamageLayers

Declaration

public int DamageLayers { get; set; }


Description

Indicates which layers can be damaged.

When the TryReleaseChargeAttack function spawns a projectile, it automatically configures the layers that the projectile can damage based on the attack's configuration.


In the example below, the spawned projectile will log the layers that can get damaged, upon collision. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private CombatSystemProjectile csProjectile;

    private void Awake()
    {
        csProjectile = GetComponent<CombatSystemProjectile>();
        
        int layers = csProjectile.DamageLayers;
        Debug.Log("I can damage layers: " + layers);
    }
}
