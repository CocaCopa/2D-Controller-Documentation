CombatSystemProjectile.InitialVelocity

Declaration

public Vector2 InitialVelocity { get; set; }


Description

The initial velocity of the projectile when it's launched.

When the TryReleaseChargeAttack function spawns a projectile, it determines the initial velocity of the projectile by considering both the current charge level of the attack and the configured parameters of the attack. The velocity is set accordingly, taking into account the charging duration and the specific characteristics defined for the attack configuration.


In the example below, the spawned projectile will log its initial velocity once it is spawned. 

using UnityEngine;

public class Example : MonoBehaviour
{
    private CombatSystemProjectile csProjectile;

    private void Awake()
    {
        csProjectile = GetComponent<CombatSystemProjectile>();
        
        Vector2 initialVelocity = csProjectile.InitialVelocity;
        Debug.Log("Initial Projectile Velocity: " + initialVelocity);
    }
}
