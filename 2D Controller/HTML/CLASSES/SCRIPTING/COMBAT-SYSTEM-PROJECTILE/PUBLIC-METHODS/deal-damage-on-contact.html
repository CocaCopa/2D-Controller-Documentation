CombatSystemProjectile.DealDamageOnContact

Declaration

public void DealDamageOnContact();


Description

Deals damage to the first collider that enters the specified hit box.

Upon executing this function, it will search for a game object that implements the IDamageable interface within the specified hitbox. Upon successfully finding such a game object, damage will be inflicted upon it.
It's worth noting that the functionality of taking damage is already handled by the EntityHealth class.


The following example illustrates how to utilize this function. 

using UnityEngine;
using CharacterController2D.CombatSystem;

[RequireComponent(typeof(CombatSystemProjectile))]
public class Example : MonoBehaviour
{
    private CombatSystemProjectile csProjectile;

    private void Awake()
    {
        csProjectile = GetComponent<CombatSystemProjectile>();
    }

    private void OnCollisionEnter2D(Collision2D collision)
    {
        csProjectile.DealDamageOnContact();
        Destroy(gameObject);
    }
}
