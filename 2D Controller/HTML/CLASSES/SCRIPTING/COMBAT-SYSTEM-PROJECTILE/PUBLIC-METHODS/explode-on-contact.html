CombatSystemProjectile.ExplodeOnContact

Declaration

public void ExplodeOnContact(bool destroyProjectile = true);


Parameters
destroyProjectile True, the game object, to which this component is attached, will be destroyed upon completion of the function's execution.

Description

Deals damage to all colliders inside the specified hitbox.

Upon executing this function, it will search for all game objects that implement the IDamageable interface within the specified hitbox. Any game object found inside said hitbox, will incur damage.
It's worth noting that the functionality of taking damage is already handled by the EntityHealth class.


The example below demonstrates a simple way to utilize this function. Once the projectile collides with an object, it will deal damage to every damageable object around the specified area. 

using UnityEngine;

[RequireComponent(typeof(CombatSystemProjectile))]
public class Example : MonoBehaviour
{
    private CombatSystemProjectile csProjectile;

    private void Awake()
    {
        csProjectile = GetComponent<CombatSystemProjectile>();
    }

    private void OnCollisionEnter2D(Collision2D collision)
    {
        csProjectile.ExplodeOnContact();
        Destroy(gameObject);
    }
}
