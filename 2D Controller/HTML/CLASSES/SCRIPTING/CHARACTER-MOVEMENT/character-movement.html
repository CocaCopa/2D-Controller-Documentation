CharacterMovement

SWITCH TO MANUAL

Dependencies

Required Component: CharacterEnvironmentalQuery


Description

This class manages your character's movement, seamlessly combining animation curves with Unity's Rigidbody2D component. This dynamic setup empowers you to create various types of movement, whether snappy or smooth.

While the script can operate independently without any required components, exercise caution when altering the movement speed or direction of your character from external sources at runtime. It's crucial to inform the script of any sudden changes that occur outside of it. Failure to do so may result in unexpected behaviour and cause your character to deviate from its intended functionality.


Public Properties
TopSpeed The character's set movement speed.
CurrentSpeed The current speed of your character.
MoveDirection The direction that your character is facing.
Public Methods
SetCharacterMoveSpeed Lets you change the default top speed of your character.
OnGroundHorizontalVelocity Calculates the velocity exclusively along the X axis. Best used when the character is grounded.
OnAirHorizontalVelocity Calculates the velocity exclusively along the X axis. Handles your character's aerial movement.
VerticalVelocity Calculates the velocity exclusively along the Y axis. Handles your character's jump.
ResetOnGroundValues Resets all values that calculate grounded-movement to 0.
ResetOnAirValues Resets all values that calculate aerial-movement to 0.
