CharacterMovement.OnGroundHorizontalVelocity

Declaration

public Vector2 OnGroundHorizontalVelocity(Vector2 moveInput);


Parameters
moveInput Player's input

Returns

A vector with the calculated X axis velocity.


Description

Calculates the velocity exclusively along the X axis. Best used when the character is grounded.

This method ensures smooth and responsive movement by dynamically adjusting the direction and speed of the character based on the given input. The precise control it offers contributes to a more seamless interaction with the game environment. Overall, it enhances the character's grounded movement for a more immersive gameplay experience.


Here is an example of how you can utilize the method: 

using UnityEngine;
using CharacterController2D.LocomotionSystem;

[RequireComponent(typeof(CharacterMovement), typeof(Rigidbody2D), typeof(CapsuleCollider2D))]
public class Example : MonoBehaviour 
{
    private Rigidbody2D characterRb;
    private CharacterMovement movement;

    private void Awake()
    {
        characterRb = GetComponent<Rigidbody2D>();
        movement = GetComponent<CharacterMovement>();
    }

    private void Update()
    {
        float horizontalInput = Input.GetAxisRaw("Horizontal");
        Vector2 vectorInput = new Vector2(horizontalInput, 0f);

        Vector2 horizontalVelocity = movement.OnGroundHorizontalVelocity(vectorInput);
        characterRb.velocity = horizontalVelocity;
    }
}
