CharacterMovement.ResetOnGroundValues

Declaration

public void ResetOnGroundValues();


Description

Resets all values that calculate grounded-movement to 0.

This method ensures that the character's movement is properly reset, eliminating residual speed and preparing the system for fresh input in the next frame.


In the example below, the function will be used when the character respawns, utilizing the EntityHealth class for convenience. 

using UnityEngine;
using CharacterController2D.LocomotionSystem;

[RequireComponent(typeof(CharacterMovement))]
public class Example : MonoBehaviour 
{
